# Dream School

Static SEO landing site for `dreamschool.com`, built with Astro, Tailwind CSS, MDX content, sitemap generation and Vercel deployment in mind.

## Run Locally

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
npm run preview
```

## Content Model

Program guide content can live in `src/content/programs/*.mdx`. The first file is:

```text
src/content/programs/x-hec-entrepreneurs.fr.mdx
```

For a new program, add a new MDX file with the same frontmatter shape, then create or map a landing page under `src/pages`. The current architecture already has French primary URLs and `/en/` mirrors wired through canonical and hreflang tags.

## SEO Checks

- Titles and descriptions are set in each Astro page.
- Canonical, Open Graph, Twitter Card and hreflang tags are generated in `src/layouts/BaseLayout.astro`.
- Sitemap is generated by `@astrojs/sitemap` during `npm run build`.
- `public/robots.txt` allows all crawlers and points to the sitemap index.
- JSON-LD is included for Organization, Article, Course, FAQPage and BreadcrumbList where relevant.

After deploy, verify:

1. Google Rich Results Test for `/`, `/faq` and `/x-hec-entrepreneurs`.
2. Google Search Console sitemap submission: `https://dreamschool.com/sitemap-index.xml`.
3. Lighthouse for Performance, SEO and Accessibility.

## Vercel Deploy

Import the repository in Vercel and use the default Astro settings:

```text
Build command: npm run build
Output directory: dist
```

Set the production domain to `dreamschool.com`.

## Brevo Wiring

Forms currently post to `/merci` for static behavior. A placeholder snippet exists at:

```text
src/integrations-brevo-placeholder.ts
```

To activate Brevo:

1. Add `BREVO_API_KEY` and `BREVO_LIST_ID` in Vercel environment variables.
2. Add the Vercel server adapter or route the forms through an external form backend.
3. Call Brevo's contacts API, then return the PDF link or redirect to `/merci`.

## Design Tokens

Colors and fonts are defined in `tailwind.config.mjs`:

- Primary navy: `#0A2540`
- Accent gold: `#D4AF37`
- Background mist: `#F7F9FC`
- Text ink: `#1A1A1A`
- Body: Inter
- Headlines: Fraunces

Global typography and article styles are in `src/styles.css`.
